Projeto de Videos

projeto construído para treinar a inserção de vídeos em nosso site.

Saiba mais...

Para realizar a inserção de vídeos em sites de forma semântica e profissional, a recomendação da W3C é a utilização da tag (video), introduzida no HTML5. De acordo com os padrões de acessibilidade e interoperabilidade, é fundamental fornecer múltiplos formatos de arquivo (como MP4 e WebM) através da tag (source) para garantir a compatibilidade entre diferentes navegadores. Além disso, as normas prezam pela experiência do usuário, sugerindo a inclusão do atributo controls para que o visitante tenha autonomia sobre a reprodução, e a utilização de faixas de texto via tag (track) para legendas, garantindo que o conteúdo seja acessível a pessoas com deficiência auditiva.

Projeto Cordel

projeto construído para treinar o efeito paralax em imagens.

Saiba mais...

O efeito parallax é uma técnica de design visual onde as imagens de fundo se movem em uma velocidade diferente dos elementos de primeiro plano (geralmente texto ou outros objetos) enquanto o usuário faz a rolagem (scroll) da página. Isso cria uma ilusão de profundidade e imersão, transformando uma interface 2D em uma experiência que simula 3D. Existem diversas formas de implementar esse efeito, desde soluções simples apenas com CSS até manipulações mais complexas com JavaScript.
